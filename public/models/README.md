# F1 Car Model Asset

Place the downloaded Formula 1 GLB at:

```text
public/models/f1-2026-car.glb
```

The builder viewer loads this local file and applies selected livery and power-unit colors directly to compatible mesh materials. If the file is missing or unavailable, the app keeps the builder usable with the built-in procedural fallback car.

Suggested source:

- FetchCFD F1 2026 Car 3D Model
- Listed license: Creative Commons Attribution 4.0
- Credit: Nimaxo, via FetchCFD project by Alges Mann

FetchCFD and Sketchfab currently require an authenticated download session for the original archive, so the binary asset is not committed unless it has been downloaded through an account with permission to access it.
